# Lectura: AI-ассистент для лекций

**Lectura** — это локальный ИИ-агент, который превращает фотографии учебных материалов, досок и рукописных конспектов в структурированные Markdown-заметки с поддержкой LaTeX.

> **Статус:** Проект в разработке. 
> На данный момент реализован модуль **Vision (OCR)** на базе модели Qwen3-vl.


## Текущие возможности (OCR Module)
1. Распознавание текста с фото и скриншотов. 
2. Автоматическое преобразование таблиц в формат Markdown.  
3. Извлечение математических формул в формат LaTeX/md \$...\$. 
4. Оптимизация изображений (авто-ресайз) для ускорения обработки. 


##  Структура проекта
- `main.py` — точка входа, координирующая обработку файлов.
- `vision.py` — логика взаимодействия с моделью зрения через Ollama.
- `config.py` — централизованные настройки, логгирование и пути.
- `photos/` — папка для загрузки исходных изображений.
- `notes/` — папка для сохранения готовых Markdown-конспектов.
